AUTHORIZATION

1. User Token

2. Client Token

POST PARAMETERS

Name Type Description
Amount The amount to recharge (required)
Phone String The receipient's phone number(required)
Email String The customer's email address(required)
Paycode String The paycode of the sunscriber network as returned from the GetBillerPaymentItems response from Step 2 (required)
Customerid String The customer Id of the receiving merchant as returned from Step 2 (required)
